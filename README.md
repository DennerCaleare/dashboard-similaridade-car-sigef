# Dashboard de Analise de Similaridade CAR-SIGEF

Dashboard interativo para analise exploratoria de dados de similaridade espacial entre registros do Cadastro Ambiental Rural (CAR) e do Sistema de Gestao Fundiaria (SIGEF).

## Caracteristicas

- Analise de similaridade espacial usando Indice de Jaccard
- Cruzamento de titularidade (CPF/CNPJ)
- Visualizacoes interativas por regiao, UF, tamanho e status
- Performance otimizada com DuckDB
- Cache inteligente para filtros
